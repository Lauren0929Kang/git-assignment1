git-assignment1


top — 시스템에서 지금 돌아가는 프로세스들을 실시간으로 보여주는 명령어입니다. CPU·메모리를 누가 많이 쓰는지 계속 갱신하면서 표시해요. 윈도우의 "작업 관리자"와 비슷합니다. 종료하려면 q를 누릅니다.


ps — process status의 약자. 지금 이 순간 실행 중인 프로세스 목록을 한 장의 사진처럼(스냅샷) 보여줍니다. top은 계속 움직이는 영상, ps는 정지 사진이라고 보면 돼요. 자주 쓰는 형태는 ps aux 또는 ps -ef이고, 프로세스의 **PID(번호)**를 확인할 때 많이 씁니다.


jobs — 지금 사용하는 터미널에서 백그라운드로 돌리거나 잠시 멈춰둔 작업 목록을 보여줍니다. 관련해서 Ctrl + Z(작업 일시정지), 명령어 뒤에 &(백그라운드 실행), fg(다시 앞으로), bg(백그라운드에서 계속) 같은 것도 같이 적으면 좋아요.


kill — 실행 중인 프로세스에 신호를 보내 종료시키는 명령어입니다. kill [PID] 형태로 쓰고, 안 죽는 프로세스는 kill -9 [PID]로 강제 종료합니다. (ps로 PID를 찾아서 kill로 죽이는 흐름이 자연스러워요.)
